Histórico

Em um período não muito distante a terra se torna inabitável, a poluição e os dejetos na superfície se tornam tão tóxicos que a humanidade se vê a beira da extinção. Os cientistas e os governos do mundo fizeram um plano audacioso para salvar a humanidade.

Criaram no espaço, imensas bases espaciais para abrigar parte da população remanescente, a fim de salvar a humanidade.

Parte da população idosa e a de pouco estudo ficou na terra encaminhando os recursos para as bases espaciais, mas com poucas décadas essa população foi extinta.

As bases espaciais se viram em uma situação complicada, necessitavam dos recursos da terra e não tinham mais pessoas para enviarem.

O Governo espacial criou um concurso para os cientistas desenvolverem em tempo recorde um sistema autômato para buscar tais recursos.

Este sistema deve utilizar os foguetes de envio que já estão na terra. Estes enviam ,de tempos em tempos, os recursos para as bases espaciais, ou seja, são foguetes autônomos que trazem para o espaço os recursos coletados. Existem vários foguetes coletores um para cada tipo de material.

Os foguetes são as unidades de guarda e transporte do materiais, a unidade de busca e coleta deve ser desenvolvida para encontrar os recursos e coletar e enviar para as unidades de guarda e transporte.

Serão envidados vários robôs exploradores, para as mais diversas áreas do planeta, estes devem reconhecer a área, encontrando os foguetes depósitos, e assim encaminhar para cada um os resíduos correspondentes.

O tão famigerado lixo de ontem é o recurso fundamental para a manutenção da vida da humanidade.


O Desenvolvimento

Crie um robô autômato para explorar uma área, contendo para isso somente 8 sensores, que conseguem identificar o ambiente nas quatro direções: frente, diagonal frontal direita, diagonal frontal esquerda, costas, diagonal posterior direita, diagonal posterior esquerda, direita e esquerda.

Este robô deve mapear esta área, encontrando os materiais que são úteis no espaço, que são: Metal, Orgânicos, Papel, Plástico e Vidro.

Deve encontrar principalmente os foguetes de transporte que estão na sua área pois a coleta só deve ser feita dos materiais que podem se transportados. Ou seja, existe um foguete para cada tipo de material, e estes podem ou não existir em uma determinada área.

Onde o robô aterrissa e sua base terrestre onde fica a unidade de recarga, contendo para tal os painéis solares. Ele deve acompanhar o estado da sua bateria para não ficar sem energia. Assim, de tempos em tempos deve-se retornar a base para completar a carga.

Deve-se acompanhar a carga da caçamba e onde ficam os foguetes de retorno dos materiais. Assim, o robô só coleta enquanto há espaço na caçamba.

As interfaces(veja em dowloads)

Para desenvolver este sistema existem duas interfaces: robo e ambiente. A primeira contém as funções que o robô deve manter para comunicação com o ambiente. A segunda contém as informações que o ambiente fornece para as decisões do robô, e também as operações que ele pode fazer no ambiente.

br.jrc.mp.Robo

public void setAmbiente(Ambiente amb) – Recebe o objeto ambiente, por onde recebera informações do mesmo e realizará as suas atividades;

public String getNome() - Retorna o nome do robo;

public void finalizar() - Recebe do ambiente a informação para finalizar as atividades.

br.jrc.mp.lib.Ambiente

public boolean andar(int direcao, byte[][] mapeamento, byte[][] mapeamentolixo) – Faz o robô andar em uma determinada direção, veja os valores estáticos do ambiente, informa o estado do mapeamento do ambiemte (na sua visão), o mapeamentolixo é a visão histórica de onde foi encontrado os elementos, o mapeamento e a visão atual, após as limpezas.;

public int ver(int direcao); Retorna uma leitura do ambiente, da direção informada, veja os valores estáticos do ambiente.

public boolean recarregarBateria() - Recarrega a bateria do robô sempre que estiver na coordenada da base;

public int verNivelBateria();

public int verNivelMaximoBateria() - Ver o tamanho máximo da bateria;

public boolean pegarLixo() - Recolhe o lixo do ambiente, desde que esteja na coordenada do lixo, retorna verdadeiro para a coleta positiva;

public boolean descarregarLixo() - Descarrega o lixo desde que esteja na coordenada do foguete de recolhimento;

public int verNivelCacamba() - Retorna a quantidade de material já recolhido;

public int verNivelMaximoCacamba() - Retorna o nível máximo da caçamba;

public void finalizar() - Informa para o ambiente que as atividade do robô finalizaram.

No Ambiente existem as constantes que devem ser utilizadas para indetificar os elementos no ambiente.

Executando

1 - No prompt de comando digite: java -jar -Xms256m -Xmx512m -XX:+UseParallelGC  jrcMapeamentoPesquisa.jar

2 - No menu Mapas clique em Abrir, escolha um mapa;

3 - No menu Robos escolha o Carregar, ou Avaliar;

4 - Clique no botão Executar.
